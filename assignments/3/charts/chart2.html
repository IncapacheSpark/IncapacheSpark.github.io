    3. Choropleth map (2)

    Tree density per Neighbourhood

    A choroplet map is a thematic map which represents the effect of some variable on a map, with some different colors. The following visualization displays the density of surveyed trees in each neighborhood of Trento and density is computed as the ratio of tree tally and area of neighbourhood. A bigger density of trees is associated to a darker shade of green, according to a linear scale.
