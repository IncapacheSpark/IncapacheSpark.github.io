    CO2 absorption compared with two size metrics

    Starting from the intuition that trees of bigger sizes should help the environment more, we created an interactive chart that sorts trees according to diverse size metrics and makes it possible to view in how much they help they environment by absorption of CO2 and attenuation of urban pollution. The x-axis hosts the size metrics (height, canopy size, leaf biomass etc) while the y-axis hosts the envornmental metrics (CO2 absorbed by the tree or pollution attenuation).
    For this task we did not use a simple scatter plot which displays data point in two dimension, but a bubble chart which takes into account a third dimension. Specifically the third dimension (z-axis) is represented as the size of the points, therefore in these circumstance data points are turned into bubbles, here the name bubble chart.
    The underlaying interactivity of the plot consists of the possibility of picking the tree metrics for the x-axis and z-axis and picking the environmental variable for the y-axis
