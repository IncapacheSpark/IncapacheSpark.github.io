    CO2 absorption compared with Tree size.

    Starting from the intuition that trees of bigger size should help the environment more, we created an interactive chart that sorts trees according to diverse size metrics and makes it possible to view how much they help they environment by absorption of CO2 and attenuation of urban pollution. The x-axis hosts the size metrics (height, canopy size, leaf biomass etc) while the y-axis hosts the envornmental metrics (CO2 absorbed by the tree or pollution attenuation).
    The user can interact with plot by picking the tree size metrics for the x-axis and by picking the environmental variable for the y-axis.
    This kind of visualization is named scatter plot and displays a collection of points, each having the value of one variable determining the position on the horizontal axis and the value of the other variable determining the position on the vertical axis.
